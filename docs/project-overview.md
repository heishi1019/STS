# 项目简介

## 项目名称

**生物医学文献知识服务与智能综述平台**。该名称与《项目案例与毕业设计方向清单》中的项目 1 一致。

## 背景与目标

医学研究者常在 PubMed、Europe PMC、Semantic Scholar 等来源重复检索，不同来源字段不统一，DOI、PMID 和标题变体又会造成重复。专题、标签、PDF 链接和阅读结论散落在多个工具中，难以沉淀为可复用的课题组资产。

本项目统一完成多源采集、标准化、去重、专题组织、检索和统计，先解决“收得进、找得到、理得清”，再为 AI 综述与证据服务提供可信数据底座。

## 用户

- 医学科研人员、研究生与导师
- 临床研究与循证医学人员
- 文献数据整理和质量检查人员

## 核心功能

1. PubMed、Europe PMC、Semantic Scholar 等多源采集与增量更新。
2. XML/JSON 解析，按 PMID、DOI、标题指纹分层去重。
3. 标题、摘要、关键词、作者、年份、期刊、DOI、PMID 组合检索。
4. 专题、标签、分类、收藏、备注及 PDF 链接管理。
5. 年度趋势、关键词词频、期刊与作者分布统计。
6. 预留带来源引用的文献问答、综述生成和证据片段定位接口。

## 首版范围与技术路线

首版以“糖尿病相关研究”为演示专题，并预置“肿瘤免疫治疗”“阿尔茨海默病”。平台模型不绑定单一疾病。

- 前端：React + TypeScript
- 后端：FastAPI
- 数据库：PostgreSQL 内置全文检索
- 采集器：Python 数据源适配器
- 部署：后续使用 Docker Compose 部署至阿里云

项目案例阶段完成完整文献管理系统；毕业设计阶段增加 AI 问答、智能综述、证据定位、热点与关键词演化，形成明确功能升级。
